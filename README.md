### Building VERDICT In Public

**[VERDICT](https://github.com/TimothyVang/verdict-dfir-community)** is DFIR for agents: a typed,
read-only, audit-chained forensic tool surface any AI agent can drive across memory, disk, EVTX,
and network evidence.

The project is built around one rule: **Don't trust the model. Reproduce the finding.** Every
Finding must cite the exact tool call that produced it, and completed runs emit signed artifacts
you can verify offline.

### What I'm Looking For

- DFIR practitioners who want agent workflows that stay evidence-bound.
- Memory forensics and parser engineers who want to improve artifact coverage.
- LLM evaluation builders who care about recall, false positives, and reproducibility.

If that sounds like you, start with the
[open problems](https://github.com/TimothyVang/verdict-dfir-community/blob/main/docs/help-wanted.md)
or open an issue on VERDICT.
