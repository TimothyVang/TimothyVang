### 🔭 What I'm building

**[VERDICT](https://github.com/TimothyVang/verdict-dfir-community)** — *DFIR for agents.* A typed,
read-only, audit-chained forensic tool surface (45 MCP tools) that any AI agent can drive to
investigate memory, disk, EVTX, and network evidence — and produce a verdict you can **verify
offline**, backed by a cryptographic chain of custody. It's built so the AI can't overclaim: every
Finding cites the exact tool call that produced it, and a deterministic, LLM-free check rejects any
value that isn't actually in the evidence.

- 🛠️ Open source (Apache-2.0) — self-host it, run it air-gapped, forever.
- 🤝 I'm looking for contributors who know **DFIR, memory forensics, or LLM evaluation** —
  [start with the open problems](https://github.com/TimothyVang/verdict-dfir-community/blob/main/docs/help-wanted.md).
